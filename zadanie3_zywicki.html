Cechy modyfikowania formatowania w HTML oraz CSS

Cechy HTML CSS
Umiejscowienie stylu Bezpośrednio w znacznikach (atrybuty jak bgcolor, font) Osobno od struktury (zewnętrzne, wewnętrzne lub inline style)
Rozdzielenie treści i wyglądu Brak Tak
Możliwość ponownego użycia stylu Ograniczona Wysoka (klasy, ID, selektory)
Elastyczność formatowania Mała Duża (layouty, animacje, media queries)

Nowe możliwości CSS w porównaniu z HTML

  1. Tworzenie zaawansowanych układów (Flexbox, Grid)
  2. Dodawanie animacji i przejść
  3. Dostosowanie stylów do różnych urządzeń (media queries)
  4. Stosowanie gradientów, cieni, efektów graficznych
  5. Lepsze zarządzanie kolorami, czcionkami i marginesami
  6. Użycie zmiennych i pseudo-klas

Wady CSS

  • Różna interpretacja przez przeglądarki (problemy z kompatybilnością)
  • Możliwość nadpisywania stylów (konflikty selektorów)
  • Brak pełnej kontroli nad wszystkimi elementami w starszych przeglądarkach
  • Może być trudne do utrzymania w dużych projektach bez dobrej struktury
